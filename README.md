# Pixora — TikTok Upload Booster

Chrome extension untuk meningkatkan kualitas upload TikTok.

## Fitur
- **Upload Boost** — aktif otomatis saat buka halaman upload
- **Prepare Video** — modifikasi file MP4 agar TikTok tidak re-encode
- **Download** — unduh video/audio TikTok tanpa watermark

## Instalasi

1. Download atau clone repo ini
2. Buka Chrome → `chrome://extensions`
3. Aktifkan **Developer mode** (pojok kanan atas)
4. Klik **Load unpacked** → pilih folder repo ini
5. Extension siap digunakan

## Cara Pakai

1. Buka [TikTok](https://www.tiktok.com) dan login
2. Klik icon Pixora di toolbar Chrome
3. Register dengan akun TikTok kamu
4. Untuk upload: buka halaman upload TikTok → boost aktif otomatis
5. Untuk prepare video: drag & drop file MP4 ke panel Pixora

## Requirements

- Google Chrome / Chromium
- Akun TikTok aktif
